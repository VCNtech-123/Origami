# Origami Project

A multi-page instructional website dedicated to the art of paper folding. This project represents my very first step into web development, built entirely with core web technologies.

## 🚀 Project Overview
This website serves as a beginner-friendly guide to origami, featuring different sections for learning, history, and contact information. The goal was to practice structuring content with HTML and creating a clean layout with CSS.

## 🛠️ Built With
- **HTML5:** Semantic structure for multiple pages (Home, About, Lessons, and Contact).
- **CSS3:** Custom styling, layout management, and typography.
- **Local Assets:** Managed custom fonts and images to enhance the visual experience.

## 📂 Features
- **Multi-page Navigation:** Seamlessly browse between different sections of the site.
- **Instructional Layouts:** Organized content in the `lessons.html` page to make learning origami easier.
- **Responsive Basics:** Simple layouts designed to be readable and clear.

## 📖 What I Learned
As my first project, I focused on:
1. Linking multiple HTML documents.
2. Understanding the CSS Box Model.
3. Managing file paths for images and local fonts.
4. Structuring forms for the contact page.
